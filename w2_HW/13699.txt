1.
t(0)=1
t(n)=t(0)*t(n-1)+t(1)*t(n-2)+...+t(n-1)*t(0)
주어진 입력 0 ≤ n ≤ 35에 대하여 t(n)을 출력하는 프로그램

2.
int 배열 t에 t[n]의 결과 값을 저장해 나가면서 해결.
t[0] = 1;
t[n] += t[0]*t[n-1]
t[n] += t[1]*t[n-2]
...
t[n] += t[n-1]*t[0]
